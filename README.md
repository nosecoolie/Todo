# [Todo](https://nosecoolie.github.io/Todo/)
A simple todo app.

### Installation
* You don't, just go [here](https://nosecoolie.github.io/Todo/)
* Or you can clone this repo and try npm install, npm run dev.

### Dependencies
* React
* Babel
* Webpack
* Use [Bulma](http://bulma.io/) for simple layout
